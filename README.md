A fun, modern take on the classic game of Tic-Tac-Toe. It features a cool glowing space theme, custom player names, and the ability to play against a smart computer or a friend on the same screen.

A highly dynamic, creative, and fully responsive Tic-Tac-Toe web application. Built with modern web technologies, this project features an ultra-modern "Cosmic Neon Glassmorphism" UI, fluid animations, and a smart AI opponent.

## ✨ Features

* **Cosmic Glassmorphism UI:** Stunning translucent cards, heavy background blurs, and deep animated cosmic gradients.
* **Neon Markers:** Glowing Cyan 'X's and Magenta 'O's bring the classic game into the future.
* **Immersive Flow:** * Animated Splash Screen logo on launch.
  * Main Menu for setting up player names and game modes.
  * Celebratory Game Over modal with glowing line-strikes for the winning sequence.
* **Two Game Modes:**
  * **1 Player (vs. AI):** Play against a smart AI algorithm that actively blocks and tries to win (not just random clicks!).
  * **2 Players (Local Co-op):** Pass and play on the same screen.
* **Session Tracking:** Keeps track of Player 1 wins, Player 2/AI wins, and draws across your current session.

## 🛠️ Tech Stack

* **Framework:** [React 18](https://react.dev/) (Functional Components & Hooks)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Build Tool:** [Vite](https://vitejs.dev/)

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You will need [Node.js](https://nodejs.org/) installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone git@github.com:rajkaleO7/Tic-Tac-Toe.git
   ```

2. **Navigate to the project directory:**


3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:5173` (or the port Vite provides in your terminal).

## 🎮 How to Play

1. Wait for the cosmic splash screen to finish loading.
2. Enter your custom player names (or leave the defaults).
3. Select your game mode: **1 Player** to test your skills against the AI, or **2 Players** for local co-op.
4. Click **Start Game**.
5. Take turns clicking the glowing grid to place your X or O. Get 3 in a row to win!
6. Use the **Play Again** button to keep your scores rolling, or **Main Menu** to reset the session.
